enum Assertation

package utest

Enumerates the states available as a result of invoking one of the static methods of @see {@link utest.Assert}.

Values

Success(pos:PosInfos)

Assertion is succesful

Parameters:

pos:

Code position where the Assert call has been executed

Failure(msg:String, pos:PosInfos)

Assertion is a falure. This does not denote an error in the assertion code but that the testing condition has failed for some reason. Ei.: Assert.isTrue(1 == 0);

Parameters:

msg:

An error message containing the reasons for the failure.

pos:

Code position where the Assert call has been executed

Error(e:Dynamic, stack:Array<StackItem>)

An error has occurred during the execution of the test that prevents futher assertion to be tested.

Parameters:

e:

The captured error/exception

SetupError(e:Dynamic, stack:Array<StackItem>)

An error has occurred during the Setup phase of the test. It prevents the test to be run.

Parameters:

e:

The captured error/exception

TeardownError(e:Dynamic, stack:Array<StackItem>)

An error has occurred during the Teardown phase of the test.

Parameters:

e:

The captured error/exception

TimeoutError(missedAsyncs:Int, stack:Array<StackItem>)

The asynchronous phase of a test has gone into timeout.

Parameters:

missedAsyncs:

The number of asynchronous calls that was expected to be performed before the timeout.

AsyncError(e:Dynamic, stack:Array<StackItem>)

An error has occurred during an asynchronous test.

Parameters:

e:

The captured error/exception

Warning(msg:String)

A warning state. This can be declared explicitely by an Assert call or can denote a test method that contains no assertions at all.

Parameters:

msg:

The reason behind the warning
