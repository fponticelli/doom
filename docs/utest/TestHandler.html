addAsync (?f:Void ‑> Void, timeout:Int = 250):Void ‑> Void

Adds a function that is called asynchronously.

Example: 

var fixture = new TestFixture(new TestClass(), "test");
var handler = new TestHandler(fixture);
var flag = false;
var async = handler.addAsync(function() {
flag = true;
}, 50);
handler.onTimeout.add(function(h) {
trace("TIMEOUT");
});
handler.onTested.add(function(h) {
trace(flag ? "OK" : "FAILED");
});
haxe.Timer.delay(function() async(), 10);
handler.execute();

Parameters:

f,

the function that is called asynchrnously

timeout,

the maximum time to wait for f() (default is 250)

returns a function closure that must be executed asynchrnously
